OpenFHQ
==========
##Get started
1. Clone repository
2. Install node.js
3. Run `npm install --global gulp coffee-script bower`
4. Run `sh init.sh`
5. Run `node index.js`

Server run at http://localhost:5000

Run `gulp watch` for watching changes and autocompile html, stylus, coffee, etc  
Stylus files with `_` prefix won't compile.
